1 - O que é HTML?

HTML é a componente base para a web permitindo a construção de websites e a inserção de novos conteúdos, como imagens e vídeos através de hipertexto.

Para escrever documentos HTML é apenas necessário, um editor de texto simples e conhecimento dos códigos que compõem a linguagem. Os códigos (tags) servem para indicar a função de cada elemento da página Web. Os tags funcionam como comandos de formatação de textos, formulários, links (ligações), imagens, tabelas, entre outros.

Os browsers (navegadores) identificam as tags e apresentam a página conforme está especificada. Um documento em HTML é um texto simples, que pode ser editado num vulgar editor de texto como por exemplo o notepad e transformado em hipertexto.

A linguagem HTML foi criada por Tim Berners Lee na década de 1990. As especificações da linguagem são controladas pela W3C (World Wide Web Consortium).

O HTML está na quinta versão HTML5.

2 - O que é um elemento HTML? Dê exemplos.

Um elemento em HTML é definido pela tag de abertura e pela tag de fecho, ficando entre as tags de abertura e fecho o conteúdo. Os elementos dizem ao browser como interpretar e estruturar o documento HTML.


Alguns exemplos de elementos com tag de abertura e tag de fecho:

-> A tag <html>, define a raíz de um documento HTML.

-> A tag <body>, define o corpo de um documento.

-> A tag <head>, define o título do documento, tipo de caracteres utilizados, estilos, acessibilidade e os mais variados metadados.


Alguns exemplos de elementos apenas com tag de abertura:

-> A tag <br>, executa uma quebra de linha.

-> A tag <hr>, traça uma linha horizontal.

-> A tag <img>, define o caminho para imbutir uma imagem na página html. O caminho pode ser para uma imagem local ou uma imagem alojada num website. Também se pode adicionar um texto alternativo caso a imagem não esteja disponível para ser visualizada.

4 - Quais os elementos base necessários para a criação de uma página HTML?

Os elementos base para a criação de uma página HTML são:

-> <!DOCTYPE html> esta tag não necessita de ser fechada e informa o browser de que este é uma página em HTML5.

-> <html> </html> todos os outros elementos estão contidos entre estas duas tags à excepção do elemento <!DOCTYPE html>

-> <body> </body> todo o conteúdo de um documento html está entre estas duas tags.

Aqui fica um pequeno exemplo de uma página simples em html com todos os elementos acima referidos. 

<!DOCTYPE html>
    <html>
        <head>
            <title>Hello World Tab</title>
        </head>
        <body>
            <h1>Hello World!</h1>
        </body>
    </html>

5 - O elemento <br> necessita de ser fechado?

O elemento <br> não necessita de ser fechado pois a sua função é executar uma quebra de linha.

6 - Para que servem os atributos? Dê um exemplo.

Os atributos fornecem informação adicional sobre os elementos.

Exemplificando a tag <a> que define um hyperlink, tem um atributo específico chamado href que define qual o URL da página para onde o link o direcciona.

Na seguinte linha de código podemos ver que o hiperlink está direccionado para o motor de busca do Google através do atributo href.

<a href="https://www.google.com/">Google Search Engine</a>

Google Search Engine

É de salientar que o atributo está incorporado na tag de entrada.
